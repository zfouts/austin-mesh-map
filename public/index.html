How it's calculated

Every prediction is a link budget over real terrain:

RX power = TX power + both antenna gains − path loss − clutter − fade margin, compared against receiver sensitivity.

Path loss: NTIA ITM (Longley-Rice v1.2.2) for links, inspection and repeater picks; two-edge Deygout diffraction for heatmaps. Terrain from AWS Terrain Tiles (~15–30 m) with 4/3-earth curvature.

Clutter: 0–18 dB per antenna end by environment, fading out above ~20 m AGL. Antennas: honest physical gains minus measured VSWR mismatch (rfindex.com).

Repeaters: maximin grid search — the pick is the spot whose worst link to any site is best.
